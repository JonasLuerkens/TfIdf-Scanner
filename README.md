# TFIDF-Scanner:

## Project description:
The TFIDF-Scanner is a tool that matches an input document to the documents of a desired corpus using TF-IDF vectorization and Cosine Similarity.
## Prerequisites and installation:
### Prerequisites:
- Python 3.8 or higher
- pip (Python Package Installer)
### Installation:
1. Clone the github project
2. Install python libraries (requirement file): pip install -r requirements.txt

## Usage:
1. Start the program with: python main.py
2. Enter the corpus you want to use. The reuters corpus is the default corpus, you can just press enter without input to use it
3. Enter text or words that you are looking for in the documents
4. Enter a percentile threshold to get all documents above this threshold
5. Inspect the provided documents by the program
6. Repeat for other texts (if desired)
7. Exit the programm with 'exit'
